# Pokedex

## Run in dev mode

1. Clone the repo
2. Install dependencies

   ```
   pnpm install
   ```

3. Install Nest CLI

   ```shell
   npm i -g @nestjs/cli
   ```

4. Deploy MongoDB instance

   ```shell
   docker-compose up -d
   ```

## Used stack

- NodeJS
- NestJS
- MongoDB
